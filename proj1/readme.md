# ML DataFrame Exploration Project

This project shows basic DataFrame exploration before machine learning.

It reads the online Titanic dataset and checks:

- shape
- sample rows
- column data types
- missing values
- missing percentage
- numeric statistics
- categorical statistics
- unique values
- duplicate rows
- target distribution
- correlation
- train/test split
- preprocessing with scaling and one-hot encoding

## Run

```bash
pip install -r requirements.txt
python main.py



## Memory line
